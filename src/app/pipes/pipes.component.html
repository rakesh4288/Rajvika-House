Pipes are simple functions you can use in template expressions to accept an input value and return a transformed value. Pipes are useful because you can use them throughout your application, while only declaring each pipe once. For example, you would use a pipe to show a date as February 04, 1988 rather than the raw string format.

  • Pipes are nothing but just a process By using them you can get a human readbale format
  • Pipes are basically filter or formatter which helps to focus format or transform raw data and display user.
Read More about Angular Pipes

Angular offers lots of built-in Pipes to deal with specific problems while developing Angular application.

Angular offers some built-in Pipes API to update your data in a quick snap. It also allows you to create custom Pipes in your app.

Currency Pipe
The currency pipe in angular helps to convert numbers in various currencies.

USD: {{ itemPrice | currency:'USD':true }}, EUR: {{ itemPrice | currency:'EUR':true}}, INR: {{ itemPrice | currency:'INR' }}

Date Pipe
The date pipe in Angular helps to format the Date in Angular.

Current Date: {{ currentDate | date: 'fullDate' }}

Get Year: {{ currentDate | date: 'yy' }}

Get Time: {{ currentDate | date: 'hm' }}

Slice Pipe
The Slice pipe API in Angular formulate a subset list or string.

Jason Pipe
The JSON pipe API works to expose an object as a JSON string in the Angular app. It complements the JSON.stringify method behind the curtains.

{{countryList | json}}

Percent Pipe
The Percent pipe API in Angular modifies a number into its percentage value.

{{myNum2 | percent}}

Lowercase & Uppercase Pipe
Lowercase or Uppercase pipes help to format text to either lower or upper case in an Angular app.

How do we call angular Pipes with multiple arguments or How can we create custom Pipes?

Well if you want to work with multiple argument in Pipes So this can be posssible with Custom Pipe
  • We can achieve this requirement by using “PipesTransform” interface and This will allow you to pass the multiple arguments.
  • Pipe class implements the “PipesTransform” interface transform method that accepts and input value and returns the transformed result.
  • To work with pipe we need to import Pipe, PipeTransform and this will be available inside ‘@angular/core’ library
  • Then we will decorate them using @Pipe()
  • name: 'custom-pipe-name' name metadata will be use for providing a name of this pipe
