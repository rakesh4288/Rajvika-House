Angular provides some binding method with element.

  • Property Binding
  • Class Binding
  • Style Binding
  • Event Binding
  • Data Binding

Property Binding:

Property binding is used to pass the data from component classs to template and setting the value of specific element.

Property Binding is an example of One-Way-Data-Binding where data is transferred from class to template

Attribute VS Property
  • Attributes and Properties are not the same
  • Attribute means - HTML
  • Properties means - DOM
  • Attribute initialize the DOM properties and they are done. Attribute values can not be change once they are initialized.
  • And property values however can be change

Class Binding:

Class binding is able to set the CSS class on a template

This is regular class using class Attribute

Using [class] binding we can pass the binded property from component class. This red color is applied by the [class] property bind

We can use the condition for [class] binding using boolean value

Style Binding:

Style binding is also used for applying the CSS But these styles are applied through the inline styling not CSS classes

using [style.color]="'orange'"

Using property value [style.color]="green"

What are the different types of data binding?
We have 4 types of data binding in Angular.
  • String interpolation using curly brasis or Expression Binding
  • Property binding
  • Event Binding like click change
  • Two Way Data Binding
To Way Data Binding:

When you are showing any data from the typescript file to html template or from html to typescript this is called two-way data binding. Like: [(ngModel)]

What are the different types of component selectors?
We have three types of Component Selectors:
  • Element Selector
  • Class Selector
  • Property Selector
